CSS Media Queries Tester

How to use this page

Parsed or Yes

These tests indicate that the media query is supported and matches the current environment.

In Parsed tests, it queries like @media (any-hover) or @media (color), which only check if the media query is recognized and parsed by the browser. If the test shows Yes, it means that the media query is supported and matches the current environment like @media (any-hover: hover) or @media (monochrome).

Unsupported / Not sure

This indicates that the media query is either not supported or the test cannot determine its support status. It may be due to the media query being unrecognized, a syntax error, or the test not being able to detect it accurately.

Please note that this might also fire when value is 0 in some case (e.g. @media (color: 0) or @media (color-index: 0)). We are trying to avoid such false negative as much as possible, but it may not be perfect. If you find any issue, please report to us with details of your environment and the test result.
